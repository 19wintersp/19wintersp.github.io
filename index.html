Autobahn

Rust
10
1
GPL-3.0

Autobahn is a remote shell, TCP tunnel, and HTTP proxy for accessing and building with Replit repls from your own computer.

Autobahn was forked from leon332157 ’s Replish (now-abandoned), and rewritten in Rust. It uses WebSockets to send extra data through Replit’s existing HTTP-only proxy.

ImageTools

Rust
3
1
GPL-3.0

ImageTools is a program for performing operations on images, for querying their properties and content, and creating simple images, from the command line.

ImageTools was created at an early time as a project for learning the Rust language. Most of the operations act as a frontend for the Rust image crate.

CFacts

0
1
MIT

CFacts is a CLI client to its own fact-of-the-day API, that can fetch daily or specific facts from a server.

CFacts was created as an example of how to write such a client in C, as well as to explore the C language. It exists more as a demonstrative affair, though does contain a minimal C JSON parser and writer.
